# React + Vite Brainstorm Board Project (Work in Progress)
This project is a simple brainstorming board application built using React and Vite. It allows users to create, edit, and delete notes on a virtual board. The application is designed to be responsive and user-friendly.

## Features
- Create new notes with a title, headings and content.
- Edit existing notes.
- Delete notes from the board.
- Drag and drop heading columns to rearrange them on the board.

## Installation
To run this project locally, follow these steps:
1. Clone the repository
2. Navigate to the project directory
3. Install the dependencies: npm install
4. Start the development server: npm run dev

## Technologies Used
- React: JavaScript library for building user interfaces.
- Vite: Build tool for modern web projects.
- TailwindCSS: For styling the application.
